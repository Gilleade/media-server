# Etapa 04 - Prowlarr

## Objetivo

Instalar e validar o Prowlarr em Docker como central de gerenciamento de indexadores e integração com aplicações da stack.

Nesta etapa, o objetivo é confirmar que:

- o container sobe corretamente;
- a Web UI abre na rede local;
- o Prowlarr consegue se conectar ao Radarr;
- a sincronização com o Radarr fica preparada.
