# Etapa 01 - Jellyfin

## Objetivo

Instalar e validar o Jellyfin em Docker como servidor inicial de mídia local.

Nesta etapa, o objetivo é confirmar que:

- o container sobe corretamente;
- a interface web abre no navegador;
- uma biblioteca local pode ser adicionada;
- um vídeo de teste pode ser reproduzido;
- o acesso funciona pela rede local.

## Serviço usado

- Jellyfin
