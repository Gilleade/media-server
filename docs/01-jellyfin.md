# Etapa 01 - Jellyfin

## Objetivo

Instalar e validar o Jellyfin em Docker como servidor inicial de mídia local.

Nesta etapa, o objetivo é confirmar que:

- o container sobe corretamente;
- a interface web abre no navegador;
- uma biblioteca local pode ser adicionada;
- um vídeo de teste pode ser reproduzido;
- o acesso funciona pela rede local.

## Serviço usado

- Jellyfin

## Porta padrão

```text
8096
```

## Status da etapa

- [x] Container iniciado
- [x] Interface web acessível
- [x] Biblioteca criada
- [x] Vídeo de teste reconhecido
- [x] Reprodução local validada
- [x] Acesso pela rede local validado

## Resultado

Etapa validada em ambiente local/intranet. O Jellyfin está funcional via Docker e acessível pelos dispositivos da rede interna.
