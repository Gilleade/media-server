# Etapa 05 - Sonarr

## Objetivo

Instalar e validar o Sonarr em Docker para gerenciamento da biblioteca local de séries.

Nesta etapa, o objetivo é confirmar que:

- o container sobe corretamente;
- a Web UI abre na rede local;
- a pasta raiz de séries é configurada;
- o Sonarr consegue se conectar ao qBittorrent;
- o Prowlarr consegue se conectar ao Sonarr.

## Serviço usado

- Sonarr
