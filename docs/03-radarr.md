# Etapa 03 - Radarr

## Objetivo

Instalar e validar o Radarr em Docker para gerenciamento da biblioteca local de filmes.

Nesta etapa, o objetivo é confirmar que:

- o container sobe corretamente;
- a Web UI abre na rede local;
- a pasta raiz de filmes é configurada;
- o Radarr consegue se conectar ao qBittorrent;
- os caminhos entre containers estão consistentes.

## Serviço usado

- Radarr
