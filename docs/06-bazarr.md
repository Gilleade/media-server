# Etapa 06 - Bazarr

## Objetivo

Instalar e validar o Bazarr em Docker para gerenciamento de legendas da biblioteca local.

Nesta etapa, o objetivo foi confirmar que:

- o container sobe corretamente;
- a Web UI abre na rede local;
- o Bazarr conecta ao Radarr;
- o Bazarr conecta ao Sonarr;
- o idioma de legenda é configurado;
- o perfil padrão de legenda é criado.

## Serviço usado

- Bazarr
